# Sound Keyboard
- [English Version 英文版本](./README.md)

## 简介
有声键盘是一个简单的软件, 此软件由Python实现, 使用到了pygame, tkinter, pynput等一些库.

## 使用方法
**1. 下载源码到一个可使用的目录**

**2. 不需要下载Python，直接运行main.exe即可**
